# iTunes-like Rating Sorting

1. Go to `Edit` > `Edit Preferences`;
2. Go to `Sorting/Grouping` section;
3. Go to `sorting` subsection;
4. Add sorting:

| `sort by:`    | `sort order:` |
|---------------|---------------|
| `Rating`      | `Descending`  |
| `Sort Artist` | `Ascending`   |
| `Sort Album`  | `Ascending`   |
| `Disc-Track#` | `Ascending`   |
